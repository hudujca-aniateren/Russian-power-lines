# Greetings
Thank you for considering using my project. Russian Electric Wires are models for Roblox games, designed for decoration in the style of the Russian provinces. They serve as decorative elements for electric wires. They can be placed both inside narrow, damp streets and on country fields and highways.

Please note that this project has been in full development since 2023. I will continue to improve it, but not at full capacity due to its low popularity. Old versions from 2022 (the start of paid releases), 2023 (the main release for everyone, also paid), 2024, and 2025 are unavailable. I foolishly lost them. But you might still see these old power lines in other games.
